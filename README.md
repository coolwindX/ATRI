### 实现
本项目可以在**任何平台**下运行，只要你具备基本的 `Python >= 3.8` 环境和一根接入互联网的网线

实现方式为 `go-cqhttp 或其它遵守Onebot标准的协议` + `NoneBot2`。

由于项目特殊性，故随机进行更新。更新日志：[传送门](changelog.md)
